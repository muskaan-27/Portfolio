My Projects

Factory Plant Location Optimizer

This tool uses Monte Carlo Simulation along with Linear Programming Optimizing Technique to find the location suitable to setup plants with different capacities. Demand, Capacity of plants, Fixed and variable costs pertaining to each country are given as input. After thorough analysis and comparison in multiple demand scenarios, the most optimal solution is provided.

Comparative Analysis of ML Models for Sentinment Analysis

In this ML-based project, we implemented various machine learning models, such as KNN and Neural Networks, to analyze and classify reviews. The project involved cleaning and preprocessing large datasets using Python, resulting in a significant reduction in data preprocessing time and an improvement in predictive model accuracy to 90%.
